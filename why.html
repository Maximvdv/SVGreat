So, why should I be using SVG?

Well, it's SVG, not science...

Introduction

In a world of modern technology, responsiveness is a key feature.
We need to customize everything for a various number of devices so our users can benefit from an optimal user experience every time.
While doing that, we also want to keep our file sizes down for performance.

With the implementation of SVG, you can keep your website pixel-perfect at all resolutions while having small file sizes and still be able to edit and modify your website.

Properties

  • Supported by all browsers
  • Resolution independant
  • Easy to edit
  • Ideal for custom animations
  • Works with CSS and Javascript
  • Highly compressible

Easy to export in files or code

SVG is usable in file-format or as raw code.
Designers can access the file to edit or improve their work.
Developers can use the raw code to create complex animations for various web applications.


Start developing your websites to its full potential, without losing performance.
